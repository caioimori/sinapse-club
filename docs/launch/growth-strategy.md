## 1. Growth Model

### 1.1 Primary Growth Loops

```
LOOP 1: Content Curation → SEO → Discovery → Sign-up
┌──────────────────────────────────────────────────┐
│ Curated AI content (50+/day)                     │
│   → Indexed by Google (PT-BR AI keywords)        │
│   → Organic traffic from search                  │
│   → Free account sign-up                         │
│   → Value experience → upgrade to Pro            │
│   → More engagement → more content → more SEO    │
└──────────────────────────────────────────────────┘

LOOP 2: Community → Social Sharing → Discovery → Sign-up
┌──────────────────────────────────────────────────┐
│ Member creates valuable post/comment             │
│   → Shares on X/LinkedIn                         │
│   → Followers click through                      │
│   → See gated content → sign up                  │
│   → New member creates content → cycle repeats   │
└──────────────────────────────────────────────────┘

LOOP 3: Course Launch → Revenue → Reinvest → More Courses
┌──────────────────────────────────────────────────┐
│ Launch course with scarcity                      │
│   → Students join community                      │
│   → Students engage in forum                     │
│   → Testimonials + case studies                  │
│   → Next launch has social proof                 │
└──────────────────────────────────────────────────┘
```

### 1.2 Flywheel

```
More curated content
  → Better SEO / more indexed pages
    → More organic traffic
      → More free users
        → More UGC posts
          → More engagement + retention
            → More Pro upgrades
              → More revenue
                → Better curation + courses
                  → More curated content (flywheel)
```

### 2.2 SEO Strategy

**Keyword strategy:**
- **Head:** "inteligencia artificial", "AI no Brasil", "cursos de AI"
- **Body:** "novidades sobre inteligencia artificial", "comunidade AI Brasil", "noticias AI portugues"
- **Long-tail:** "como usar Claude Code em portugues", "melhor modelo de linguagem 2026", "como aprender AI do zero"

**Tatica:**
Cada item de conteudo curado e traduzido gera uma pagina indexavel com:
- Title tag otimizado (PT-BR)
- Meta description
- Structured data (Article schema)
- Internal linking entre posts relacionados
- tsvector fulltext search

**Volume estimado:**
- 50 items curados/dia = 1,500 paginas novas/mes
- Em 6 meses: 9,000 paginas indexadas
- Target: 10,000+ organic visits/month em 6 meses
