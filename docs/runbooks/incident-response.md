# Runbook: Resposta a Incidente de Segurança

## SLA de Notificação
- Titulares de dados afetados: notificar em até **72 horas** após confirmação do incidente
- ANPD: notificar conforme Resolução CD/ANPD nº 15/2023

## Processo
1. **Detecção** — identificar escopo e dados afetados
2. **Contenção** — revogar acessos, isolar sistemas afetados
3. **Avaliação** — quantificar usuários impactados, tipo de dado exposto
4. **Notificação** — email para usuários afetados + formulário ANPD
5. **Remediação** — patch, auditoria, relatório final
